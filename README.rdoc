== <b>Year Up Puget Sound - Student Tracker</b>

<b>Description:</b>
A web application developed using the Ruby on Rails framework. Its purpose is to aid Year Up Puget Sound in tracking student loaned laptops and issues that may occur during a student's internship phase. It is also being used as testing material to aid the program's Quality Assurance course.

<b>Requirements</b>
- Ruby 2.0.0
- Rails 4.0.1
- PostgreSQL
- Heroku Server w/PostMark and PostgreSQL Plugins.
